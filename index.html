Integrazione Google Sheets

Collegare i dati live

Invece di caricare CSV manualmente ogni volta, punta la dashboard a un Google Sheet aggiornato dai responsabili di ogni sistema.

1
Crea 3 Google Sheet separati
Un file per Oracle/PRM, uno per FOX, uno per ServiceNow. Incolla l'export CSV in ogni sheet con la prima riga come intestazione dei campi.
2
Pubblica ogni foglio come CSV
In ogni Sheet: File → Condividi → Pubblica sul Web. Seleziona il foglio specifico, formato CSV, clicca Pubblica. Copia il link generato.
https://docs.google.com/spreadsheets/d/ ID_FOGLIO/pub?gid=0&single=true&output=csv
3
Incolla i 3 URL nella dashboard
Nella sezione "Collega Google Sheet" della dashboard incolla i link. La dashboard scarica i CSV direttamente nel browser — nessun backend richiesto.
4
Pubblica su GitHub Pages
Carica index.html e dashboard.html su GitHub. Attiva Pages in Settings → Pages → main branch. I colleghi aprono il link e usano i propri URL Sheet.
git add . git commit -m "deploy dashboard" git push origin main # Settings → Pages → Deploy from branch
5
Aggiornamento senza friction
Il responsabile aggiorna l'export nel Google Sheet. Il collega ricarica la dashboard — i dati freschi sono già disponibili. Zero upload manuali.
sheets-loader.js
